# Color Detection and Assistive Device Detection using Python

## Project Description
This project involves developing a Python application that detects colors and assists in identifying devices using computer vision techniques. The application aims to provide a user-friendly interface for individuals who may have visual impairments, helping them recognize colors and devices in their surroundings.

## Features
- Color detection in real-time using webcam.
- Identification of assistive devices and their functionalities.
- User-friendly graphical interface.
- Supports multiple color formats (RGB, HSV).
- Saves detected colors and devices in a log file.

## File Descriptions
- `main.py`: The main script that runs the application.
- `color_detector.py`: Contains functions for color detection.
- `device_recognition.py`: Functions for recognizing assistive devices.
- `requirements.txt`: List of Python packages required to run the project.

## Usage Guide
1. Run the application:
   ```
   python main.py
   ```
2. Follow the on-screen instructions to start detecting colors and devices.
3. The detected colors will be displayed, and devices will be identified as per their functionality.
4. For help, refer to the documentation or contact the project maintainer.
